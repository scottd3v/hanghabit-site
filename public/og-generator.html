A little hang goes a long way.
A minute a day keeps the back pain away.

To capture this as a PNG:

1. Open Chrome DevTools (Cmd+Option+I)

2. Click the device toolbar icon (Cmd+Shift+M)

3. Set dimensions to exactly 1200 × 630

4. Right-click → "Capture screenshot"

5. Save as og-hang-habit.png in the public folder
